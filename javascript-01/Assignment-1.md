### 4. What is ECMAScript?
 ECMAScript is a standardized scripting language specification that forms the basis of JavaScript. It defines the core features and syntax of the language, which all JavaScript engines implement.

* In **1997**, JavaScript was standardized as **ECMAScript** by Ecma International.
* **ES6 (ECMAScript 2015)** was a major update with modern features like `let`, `const`, arrow functions, classes, etc.

### 10. What is meant by interpreted and compiled programming languages? Give examples for each.
  
- Interpreted languages: Executed line-by-line by an interpreter (e.g., JavaScript, Python).  
- Compiled languages: Translated into machine code before execution by a compiler (e.g., C, C++).

### 18. What is a database? Why is it used?
 A database is an organized collection of data stored electronically. It is used to store, retrieve, and manage data efficiently.

---

### 19. What is the difference between a server and a database?

### 20. Explain the client-server architecture.
 In this architecture, clients (users' devices) send requests to servers, which process these requests and return the appropriate data or responses.

---

### 21. What happens when you type a URL in a browser?
 The browser sends a request to the web server hosting the website, retrieves the web page, and displays it to the user.

---

### 22. What is a client in a client-server model?
 The client is the device or software (like a web browser) that requests services or resources from a server.

---

### 23. What is the role of a browser in accessing the web?
